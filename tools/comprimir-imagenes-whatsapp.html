Cómo comprimir imágenes para WhatsApp

1

Sube tu imagen

Arrastra y suelta cualquier foto JPG, PNG o WebP, o haz clic para seleccionarla desde tu dispositivo.

2

Ajusta o usa el botón WhatsApp

Usa el deslizador para ajustar la calidad manualmente, o pulsa è«Tamaño WhatsAppç» para optimizar automáticamente al formato ideal (JPG, calidad 70%).

3

Descarga y comparte

Haz clic en descargar y tu imagen comprimida estará lista para enviar por WhatsApp en segundos.

Preguntas frecuentes sobre WhatsApp y compresión

WhatsApp permite enviar imágenes de hasta 16 MB como documento. Sin embargo, cuando envías una imagen desde la galería (como foto normal), WhatsApp la comprime automáticamente, a menudo reduciendo mucho la calidad. Nuestra herramienta te permite comprimirla tú mismo antes de enviarla, manteniendo el control sobre la calidad final.

WhatsApp comprime las fotos para ahorrar datos y espacio en sus servidores, lo que puede reducir notablemente la calidad. Para evitarlo tienes dos opciones: 1) Enviar la foto como Documento (icono de clip — ?Documento) — así WhatsApp no la comprime. 2) Usar nuestra herramienta para comprimir la imagen tú mismo antes de enviarla, consiguiendo un equilibrio perfecto entre tamaño y calidad.

Recomendamos usar JPG con calidad 70%. Este ajuste suele dar un archivo de aproximadamente 300-800 KB para una foto típica de móvil, que es el tamaño ideal para WhatsApp: no ocupa demasiado, se envía al instante, y la pérdida de calidad es casi imperceptible. Si la imagen tiene texto o gráficos, prueba con PNG a calidad 80%.
