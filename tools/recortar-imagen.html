Guía de recorte de imágenes

🎯 Recortar para redes sociales

Cada plataforma social tiene sus proporciones ideales. Con esta herramienta puedes aplicar presets rápidos: 1:1 para Instagram, 16:9 para YouTube o 9:16 para TikTok e historias de Instagram. Ahorra tiempo y logra un encuadre perfecto sin necesidad de programas complicados.

💡 Consejo: Usa la proporción 1:1 para fotos de perfil en cualquier red social. Es la más versátil.

📱 Fotos de perfil y avatares

Las fotos de perfil suelen mostrarse en formato cuadrado. Con el preset 1:1 puedes centrar tu rostro o el elemento principal y recortar exactamente lo que necesitas. Asegúrate de dejar algo de espacio alrededor del sujeto para que no quede demasiado ajustado.

💡 Para LinkedIn y Facebook, el recorte cuadrado funciona mejor si el rostro ocupa aproximadamente el 60% del encuadre.

🖼️ Composiciones y collages

Cuando preparas varias imágenes para un collage, es importante que todas tengan la misma proporción. Usa el preset 4:3 —es el formato clásico de fotografía— para mantener consistencia visual en tus composiciones.

📹 Miniaturas para YouTube y vídeos

Las miniaturas de YouTube se ven mejor en 16:9. Recorta tus imágenes a esta proporción antes de añadir texto o superposiciones. El encuadre horizontal destaca mejor los paisajes, productos o composiciones con texto.

💡 Al recortar para miniatura, deja espacio en los bordes para que el texto sea legible incluso en vista reducida.

🌅 Recorte libre para máxima flexibilidad

El modo "Libre" te permite recortar con cualquier proporción que necesites. Ideal para composiciones creativas, recortes irregulares o cuando necesitas eliminar bordes no deseados sin restricciones. Arrastra las esquinas hasta lograr el encuadre deseado.

🔒 Privacidad y velocidad: todo en tu navegador

Como todas las herramientas de ComprimeFotos, el recorte se procesa íntegramente en tu navegador. Tus imágenes nunca se suben a ningún servidor. Esto no solo garantiza tu privacidad —también hace que el proceso sea instantáneo, sin tiempos de espera ni límites de uso.
