Cómo redimensionar una imagen

1

Sube la imagen

Arrastra cualquier imagen JPG, PNG o WebP o haz clic para seleccionarla.

2

Elige el tamaño

Ingresa el ancho y alto en píxeles, activa «Mantener proporción» o elige un preset para redes sociales.

3

Descarga el resultado

Previsualiza el resultado y descarga tu imagen redimensionada con un clic.

Preguntas frecuentes

Al reducir el tamaño (hacer más pequeña) se pierden píxeles pero la calidad visual se mantiene bien. Al agrandar una imagen pequeña puede verse pixelada porque se están duplicando píxeles. Recomendamos siempre partir de una imagen de buena resolución.

Significa que la imagen se redimensiona sin deformarse: si cambias el ancho, el alto se ajusta automáticamente para mantener la relación de aspecto original. Así tu imagen no se ve estirada ni aplastada.

Soporta JPG, PNG y WebP. La imagen redimensionada se descarga en el mismo formato que subiste (excepto WebP que se descarga como PNG por compatibilidad).

No. Todo el procesamiento ocurre 100% en tu navegador. Tus imágenes nunca se suben a ningún servidor ni salen de tu dispositivo.

No hay un límite estricto, pero imágenes muy grandes (más de 10.000 píxeles de lado) pueden ralentizar el navegador. Para la mayoría de casos otos de redes sociales, web, etc.— funciona perfectamente.
